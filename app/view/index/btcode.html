安装宝塔

Centos 安装脚本 
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh
Ubuntu/Deepin 安装脚本 
wget -O install.sh http://download.bt.cn/install/install_6.0.sh && bash install.sh
一键更新脚本 
curl http://download.bt.cn/install/update6.sh|bash

常用管理命令 停止/启动/重启/卸载 
/etc/init.d/bt stop    # 停止
/etc/init.d/bt start   # 启动
/etc/init.d/bt restart # 重启
查看面板默认信息（地址、账号、密码） 
/etc/init.d/bt default
修改面板端口 (例：改为8881) 
echo '8881' > /www/server/panel/data/port.pl && /etc/init.d/bt restart
修改面板密码 (例：改为123456) 
cd /www/server/panel && python tools.py panel 123456
消除登录限制/域名绑定/SSL 
rm -f /www/server/panel/data/*.login        # 清理登录限制
rm -f /www/server/panel/data/domain.conf    # 解除域名绑定
rm -f /www/server/panel/data/ssl.pl && /etc/init.d/bt restart # 关闭面板SSL

Nginx 服务管理

基本操作 
/etc/init.d/nginx start   # 启动
/etc/init.d/nginx stop    # 停止
/etc/init.d/nginx restart # 重启
/etc/init.d/nginx reload  # 重载配置
关键路径 
安装目录：/www/server/nginx
配置文件：/www/server/nginx/conf/nginx.conf
站点配置：/www/server/panel/vhost/nginx

PHP 服务管理

通用命令格式 (需将 {version} 替换为具体版本，如 56, 72, 74, 80) 
# 启动
/etc/init.d/php-fpm-{version} start

# 停止
/etc/init.d/php-fpm-{version} stop

# 重启
/etc/init.d/php-fpm-{version} restart

# 重载配置
/etc/init.d/php-fpm-{version} reload
配置文件路径示例 (version=74) 
/www/server/php/74/etc/php.ini
